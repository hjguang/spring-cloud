项目说明
======
指定项目编译可以使用以下命令
cd spring-cloud
spring-cloud>mvn install -pl project-name #多项目用“,"分割

| 项目                      | 端口   | 介绍       |
|:------------------------|:-----|:---------|
| microservice-rest-movie | 8001 | movie微服务 |
| microservice-rest-user  | 8002 | user微服务  |
| microservice-gateway    | 8000 | 网关       |
